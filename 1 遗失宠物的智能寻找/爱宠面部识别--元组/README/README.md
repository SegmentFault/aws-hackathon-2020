# 项目介绍：

本项目是一个猫脸识别的案例，通过已经训练好的模型检测已存在的猫咪图片，用卷积神经网络做对图片做回归算法，提其特征，获取猫脸的花纹的主要关键点，识别出是哪一只猫，本模型可以用于走失小猫的检测与寻找。
