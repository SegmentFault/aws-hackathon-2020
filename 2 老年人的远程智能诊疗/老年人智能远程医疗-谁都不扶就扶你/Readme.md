#老年人

#应用场景介绍 老年人不方便去医院的情况时有发生，我们团队的远程智能医疗，可以通过疾病分类选择不同的疾病，输入基础信息后将不同的病人进行分类，让医生和用户更方便的在线问诊，方便快速的使患者尽快得到最好的建议。
